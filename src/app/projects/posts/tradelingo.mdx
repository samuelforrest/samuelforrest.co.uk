## Project Motivation

Gen Z faces a financial literacy crisis. While 67% want to invest, only 23% feel confident doing so. Tradelingo bridges this gap through risk-free, gamified learning.

## The Problem

Young people struggle with investing because:

- **Fear of losing money** prevents experimentation
- **Complex terminology** creates barriers
- **Boring educational content** fails to engage
- **No safe practice environment** exists

## The Solution: Tradelingo

A mobile-first stock simulator that makes learning investing:

- **Risk-free**: Use virtual money

- **Social**: Compete with friends

## Technical Architecture

### Mobile App

Built with React Native for cross-platform compatibility:

```javascript
// Example: Real-time stock price updates
const useStockPrice = (symbol) => {
  const [price, setPrice] = useState(null);

  useEffect(() => {
    const ws = new WebSocket("wss://api.tradelingo.com");
    ws.onmessage = (event) => {
      const data = JSON.parse(event.data);
      if (data.symbol === symbol) {
        setPrice(data.price);
      }
    };
    return () => ws.close();
  }, [symbol]);

  return price;
};
```
