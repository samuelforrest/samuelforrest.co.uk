## Hackathon Victory

My teammate and I won a **£300 prize** at the prestigious KTHack Hackathon with Verda, an app designed to make sustainability engaging and rewarding.

## The Problem

Climate change requires individual action, but most people:

- Don't know where to start
- Lack motivation to change habits
- Can't visualize their impact
- Find sustainability apps boring

## Our Solution: Verda

Verda gamifies sustainable living through:

### 1. Carbon Tracking

- Real-time carbon footprint calculation
- Transport method tracking
- Food consumption monitoring
- Energy usage insights

### 2. Gamification

- Points for eco-friendly actions
- Achievement badges
- Leaderboards
- Challenge system

### 3. Community

- Share progress with friends
- Team challenges
- Local sustainability events
- Impact visualization

## Technical Implementation

### Tech Stack

- **Frontend**: React Native (cross-platform)
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **APIs**: Carbon calculation APIs
- **Auth**: Firebase

### Key Features

```javascript
// Example: Calculate carbon footprint
const calculateFootprint = (transport, distance) => {
  const emissionFactors = {
    car: 0.21, // kg CO2 per km
    bus: 0.089,
    train: 0.041,
    bike: 0,
  };
  return distance * emissionFactors[transport];
};
```
